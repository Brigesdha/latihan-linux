* File & Directory Operations
  - ls -la = List all files and directories (include hidden) in long format.
  - cd <dir> = Change current working directory to <dir>
  - pwd = Print the current working directory path
  - mkdir <dir> = Create a new directory
  - mkdir -p <dir/subdir> = Create nested directories
  - rm <file> = Delete a file
  - rm -rf <dir> = Forcefully remove a directory and its contents recursively
  - cp <src> <dest> = Copy files or directories (use -r for directories)
  - mv <src> <dest> = Move or rename files
  - touch <file> = Create a new empty file
  - ln -s <file> <link> = Create a symbolic link to a file

* Viewing and Searching FIles
  - cat [file] = View file contents
  - less [file] = View file contents one page at a time
  - head -n [num] [file] = View the first [num] lines of a file
  - tail -f [file] = view the last 10 lines of a file in real-time
  - grep [pattern] [file] = Search for a pattern in a file
  - grep -r [pattern] [dir] = Search for a pattern recusively in a directory

* File Permissions and Ownership
  - chmod [permissions] [file] = Change file permissions (e.g., chmod 755 file)
  - chown [user]:[group] [file] = Change file owner and group
  - chgrp [group] [file] = Change file group ownership

* Compression and Archiving
  - tar -cvzf <archive.tar.gz> <files> = Create a gzipped tar archive
  - tar -xvzf <archive.tar.gz> = Extract a gzipped tar archive
  - zip <archive.zip> <files> = Create a zip archive
  - unzip <archive.zip> = Extract a zip archive

* User Management
  - sudo useradd <username> = Create a new User
  - sudo passwd <usernama> = Change a user's password
  - sudo usermod -aG <group> <user> = Add a user to a group
  - whoami = Display current user

* System Information and Monitoring
  - uname -a = Print system information
  - uptime = Display how long the system has been running
  - df -h = Show disk space usage in human-readable format
  - du -sh [dir] = Estimate space usage of a directory
  - free -m = Show free and used memory in MB
  - top = Monitor active system processes in real-time
  - htop = Interactive process viewer (alternative to top)
  - ps aux = Display all running processes
  - kill [pid] = Terminate a process by ID

* Networking
  - ping [host] = Send ICMP echo request to a host
  - ifconfig = View network interface configuration
  - ip addr show = Display IP addresses and interfaces
  - netstat -tuln = Show listening TCP/UDP ports
  - ssh [user]@[host] = Log in to a remote machine
  - scp [file] [user]@[host]:[path] = Securely copy files to a remote host
  - curl [URL] = Transfer data from or to a server

* Package Management (Fedora with dnf)
  - sudo dnf update = Update all installed packages
  - sudo dnf install [package] = Install a new package
  - sudo dnf remove [package] = Unistall a package
  - dnf search [keyword] = Search for a package in repositories
  - dnf list installed = List all instaled package
  - sudo dnf clean all = Clear the DNF cache

* Terminal Shortcuts
  - Ctrl+C = Stop the current command
  - Ctrl+Z = Suspend the current process
  - Ctrl+L = Clear the screen
  - Ctrl+R = Search command history
  - !! = Repeat the last command
  - exit = Log out of the current session
  - man [command] = View the manual page for a command
